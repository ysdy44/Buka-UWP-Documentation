## 什么是UWP（Universal Windows Platform）？
一言以蔽之，UWP就是在手机、平板、PC等不同(运行win10的)设备上提供同样的体验的一种通用应用程序。
这意味着在电脑的屏幕上，当你拖动应用程序的窗口使之变窄时，通用应用程序能够呈现出手机的界面布局；
而当你拖动应用程序的窗口使之变宽时，通用应用程序又会变为PC平台上的宽屏布局。
